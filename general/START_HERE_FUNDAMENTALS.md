## üî• COMMON MISTAKES TO AVOID

### **Lists**
- ‚ùå Modifying while iterating
- ‚ùå Confusing shallow vs deep copy
- ‚úÖ Use .copy() carefully with nested lists

### **Dictionaries**
- ‚ùå Accessing non-existent keys directly
- ‚ùå Forgetting to use .get()
- ‚úÖ Always use .get(key, default)

### **Sets**
- ‚ùå Trying to add unhashable objects (lists)
- ‚ùå Expecting ordered results
- ‚úÖ Remember sets are unordered!

### **Lambda**
- ‚ùå Making lambda too complex
- ‚ùå Forgetting parentheses
- ‚úÖ Use functions for complex logic

### **Comprehensions**
- ‚ùå Wrong variable names
- ‚ùå Confusing iteration order
- ‚úÖ Test nested comprehensions carefully
